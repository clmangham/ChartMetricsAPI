ChartMetricsAPI

About the API

This API provides access to medical chart data, allowing for querying individual records and aggregating data across various dimensions.

Endpoints

Record Lookup: Retrieve data for a comma-separated list of IDs.

GET /records/

Data Summary (SQL): Aggregate data using SQL queries.

GET /summary/sql

Data Summary (Pandas): Aggregate data using Pandas in Python.

GET /summary/pandas

Getting Started

To start using the API, you can make HTTP requests to the above endpoints. For example:

GET /records/?Ids=1,2,3

Please refer to the documentation for detailed information on request and response formats.
